Minha Biblioteca: Uma Webpage Personalizada

Este projeto é uma página web desenvolvida com o objetivo de compartilhar uma seleção dos meus manhwas favoritos. Cada obra listada na página conta com informações relevantes, como o nome do autor, a data de publicação e um link direto para compra na Amazon, facilitando o acesso dos visitantes que desejarem adquirir os títulos. A estrutura da página foi construída utilizando HTML e aprimorada com CSS para garantir uma apresentação visual harmoniosa e agradável.

Decidindo o Futuro: Uma Jornada Interativa sobre a Inteligência Artificial

O projeto é um jogo interativo em formato web que convida os usuários a vivenciar uma narrativa imersiva sobre os impactos da Inteligência Artificial na sociedade. Através de escolhas feitas ao longo da história, cada jogador molda o rumo dos acontecimentos, experimentando diferentes cenários possíveis em um futuro influenciado por avanços tecnológicos. Mais do que entretenimento, o jogo propõe uma reflexão profunda sobre os dilemas éticos e sociais que envolvem o uso da IA.

Explorando o Universo: Uma Aventura Interativa em Astronomia com Scratch

Este projeto desenvolvido na plataforma Scratch oferece uma experiência interativa e educativa voltada à astronomia. Por meio de cenários dinâmicos e diálogos informativos, os usuários são guiados em uma jornada de descobertas sobre temas como constelações, eclipses e a forma da Terra. A proposta combina elementos visuais envolventes com interações simples e intuitivas, tornando o aprendizado acessível e divertido, especialmente para o público infantojuvenil.
